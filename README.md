# ⌚ Horologium – Luxury Watch Studio (Git Workflow Inspired)

A visually stunning and interactive **Luxury Watch Collection Studio** inspired by **Git workflow concepts**.
This project blends **UI design + JavaScript logic + Git concepts** into a unique experience where users can *stage, commit, and push watches* just like code.

---

## 🚀 Features

* 🛍️ Explore luxury watches in a modern UI
* ➕ Add watches to **Staging Area**
* 🧠 Git-inspired workflow (Stage → Commit → Push)
* 📝 Custom commit messages
* 📜 Commit history tracking
* 🖥️ Terminal-style Git log output
* 🧹 Clear staging area
* 🎨 Premium glassmorphism UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Glassmorphism UI)
* JavaScript (Vanilla JS)

## 🎮 How It Works

* Browse available watches
* Click ➕ to add them to staging
* Write a commit message
* Click **Commit** → saved in history
* Click **Push** → simulate pushing to remote
* Watch terminal logs update like real Git
